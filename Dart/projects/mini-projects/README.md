# Dart Mini Projects

## Project Ideas
- Expense tracker CLI
- Habit and streak tracker
- Inventory or library manager

## Expected Skills
- Typed models
- Validation helpers
- Collection processing
- Async persistence simulation
- Repository-style separation

## Recommended Folder Structure
- `bin/` for entrypoints
- `lib/models/` for typed classes
- `lib/services/` for business logic
- `lib/repositories/` for storage abstractions

## Stretch Goals
- Add file-based persistence.
- Add generic result wrappers.
- Add stream-based progress updates for longer operations.
