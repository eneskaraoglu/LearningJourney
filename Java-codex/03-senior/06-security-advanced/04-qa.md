# Q&A

Q: What is OIDC?
A: OpenID Connect adds identity on top of OAuth2.

Q: Why use PKCE?
A: It protects public clients from code interception.

Q: JWT vs opaque token?
A: JWT is self-contained; opaque needs server validation.

Q: What is mTLS?
A: Mutual TLS where both sides present certificates.

Q: Why rotate secrets?
A: To reduce exposure and limit leaked credential impact.

Q: What is least privilege?
A: Grant only the minimal permissions required.

Q: What is token revocation?
A: Invalidating a token before expiration.

Q: How do you secure internal APIs?
A: Use mTLS, network policies, and strong auth.
