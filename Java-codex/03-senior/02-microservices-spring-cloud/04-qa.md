# Q&A

Q: What is a config server?
A: Centralized configuration management for distributed services.

Q: Why use an API gateway?
A: Centralize auth, routing, rate limiting, and monitoring.

Q: What is client-side load balancing?
A: The client chooses a service instance from discovery.

Q: What is a circuit breaker?
A: A guard that stops repeated calls to a failing service.

Q: What is service discovery?
A: A registry that tracks running service instances.

Q: Why are timeouts critical?
A: They prevent cascading failures and thread exhaustion.

Q: What is contract testing?
A: Validating that producers and consumers match expectations.

Q: When not to use microservices?
A: For small teams or tightly coupled domains.
