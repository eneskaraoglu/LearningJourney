# Senior Deep Exercises

3. Design a saga for order fulfillment with compensations and timeouts.

6. Propose a caching and invalidation policy for a shared catalog service.
7. Design a zero-downtime deployment strategy with rollback.
8. Prepare a 30-minute system design interview for a real-time feed.
