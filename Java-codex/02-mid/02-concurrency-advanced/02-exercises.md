# Exercises

1. Implement a thread-safe counter using `AtomicInteger`.
2. Convert a callback-based workflow to `CompletableFuture` chaining.
3. Detect and explain a deadlock in a two-lock example.
4. Build a rate limiter using `Semaphore`.
5. Compare `synchronized` with `ReentrantLock` in a scenario.
6. Implement a producer-consumer with a `BlockingQueue`.
