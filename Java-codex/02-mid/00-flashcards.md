# Mid Flashcards

Q: Heap vs stack?
A: Heap stores objects; stack stores method frames.

Q: `volatile` guarantees?
A: Visibility of changes across threads.

Q: `@WebMvcTest` is used for?
A: Controller slice tests with mocked dependencies.

Q: What is optimistic locking?
A: Version-based conflict detection for updates.

Q: JWT validation steps?
A: Verify signature, expiration, and claims.

Q: What is idempotency?
A: Safe repeated processing without side effects.

Q: Why use Actuator?
A: Health, metrics, and diagnostics endpoints.

Q: What is a correlation ID?
A: An ID to trace a request across services and logs.
