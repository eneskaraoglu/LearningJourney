# Solutions (Short)

1. Add a properties class and enable with `@EnableConfigurationProperties`.
2. Use `application-local.yml` and `application-ci.yml`.
3. Mock service with `@MockBean` and verify JSON with MockMvc.
4. Use an in-memory database and `TestEntityManager` or repository.
5. Use full context for integration behavior only.
6. Create `@TestConfiguration` with a stub bean.
