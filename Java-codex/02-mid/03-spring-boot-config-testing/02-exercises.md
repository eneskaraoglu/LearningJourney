# Exercises

1. Create a config class using `@ConfigurationProperties`.
2. Add profile-specific values for local and CI.
3. Write a `@WebMvcTest` for a controller with mocked service.
4. Write a `@DataJpaTest` that verifies a repository query.
5. Use `@SpringBootTest` only where needed and justify why.
6. Add a test configuration bean for a fake external client.
