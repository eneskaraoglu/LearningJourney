# Solutions (Short)

1. Create a `@RestController` with `@GetMapping("/health")`.
2. Use `@PostMapping`, `@GetMapping`, `@PutMapping`, `@DeleteMapping`.
3. Controller calls service; service calls repository; repository manages storage.
4. Use `spring.profiles.active=dev` and profile-specific overrides.
5. Add `spring-boot-starter-validation` and annotate DTO fields.
6. Use `ResponseEntity` and exception handlers for status mapping.
