# Beginner Flashcards

Q: What does `public static void main(String[] args)` represent?
A: The JVM entry point for a Java application.

Q: Why is `String` immutable?
A: For safety, caching, and predictable behavior.

Q: When use `equals` instead of `==`?
A: To compare object content rather than references.

Q: What is autoboxing?
A: Automatic conversion between primitives and wrapper classes.

Q: Difference between `List` and `Set`?
A: `List` allows duplicates; `Set` does not.

Q: Why use `try-with-resources`?
A: To automatically close resources.

Q: What does `@RestController` do?
A: Maps HTTP requests and returns JSON responses.

Q: What is `@Transactional` used for?
A: To define a unit of work with commit/rollback semantics.
