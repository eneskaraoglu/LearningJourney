# Exercises

1. Run three tasks in parallel using `ExecutorService` and wait for results.
2. Create a method that counts primes in a range using threads.
3. Use Streams to filter and sort a list of `User` by age.
4. Convert a list of strings to uppercase and join with commas.
5. Find the maximum value in a list using `reduce`.
6. Explain why a parallel stream might be slower on small lists.
