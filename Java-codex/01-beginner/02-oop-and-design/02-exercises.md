# Exercises

1. Create a `BankAccount` class with `deposit`, `withdraw`, and balance checks.
2. Model `Vehicle` as an interface and implement `Car` and `Bike`.
3. Implement a base `Shape` class and subclasses `Circle` and `Rectangle`.
4. Override `equals` and `hashCode` for a `User(id, email)` class.
5. Replace inheritance with composition in a `ReportPrinter` example.
6. Write an immutable `Money` value object.
