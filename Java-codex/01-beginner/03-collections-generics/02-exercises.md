# Exercises

1. Given a list of strings, return a list of unique values in insertion order.
2. Count word frequencies from a sentence using a `Map`.
3. Implement `firstNonRepeatedChar(String s)` using a `LinkedHashMap`.
4. Write a generic `Box<T>` class with `get` and `set`.
5. Convert a `List<Integer>` to a `Set<Integer>` without duplicates.
6. Explain which collection you would use for an LRU cache and why.
