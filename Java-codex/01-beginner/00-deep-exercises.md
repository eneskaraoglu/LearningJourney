# Beginner Deep Exercises

1. Build a CLI task manager with commands: `add`, `list`, `done`, `delete`.
2. Implement a `Person` CSV importer with validation and error reporting.
3. Write a `SimpleCache<K,V>` with LRU eviction using `LinkedHashMap`.
4. Create a file-based notes app using `java.nio` and JSON lines.
5. Build a small multi-threaded word counter for a directory of files.
6. Create a Spring Boot REST API for `Book` with validation and DTOs.
7. Add a `@ControllerAdvice` to map exceptions to HTTP responses.
8. Add pagination and sorting to a list endpoint.
