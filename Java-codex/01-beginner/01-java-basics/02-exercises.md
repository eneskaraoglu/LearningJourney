# Exercises

1. Write a `HelloJava` class that prints your name and the current year.
2. Read two integers from command-line args and print their sum and product.
3. Implement FizzBuzz for numbers 1 to 100.
4. Write a method `reverse(String s)` without using library reverse helpers.
5. Given an `int[]`, compute min, max, and average.
6. Build a simple calculator with `+`, `-`, `*`, `/` using `switch`.
