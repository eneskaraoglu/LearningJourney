# Solutions (Short)

1. Create a class with `main` and `System.out.println`.
2. Use `Integer.parseInt(args[0])` and `args[1]` with basic arithmetic.
3. Use `% 3` and `% 5` checks, print the number otherwise.
4. Convert to `char[]`, swap from both ends, return new `String`.
5. Track `min`, `max`, `sum` in a loop; average is `sum / (double) n`.
6. Use `switch (op)` with default for invalid operators.
