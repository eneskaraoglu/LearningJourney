# Exercises

1. Create `User` and `Order` entities with a one-to-many relation.
2. Implement repository methods for `findByEmail` and `findByStatus`.
3. Add pagination and sorting to a list endpoint.
4. Demonstrate a lazy loading issue and explain the fix.
5. Add `@Transactional` to a service method that updates two entities.
6. Write a JPQL query for orders within a date range.
