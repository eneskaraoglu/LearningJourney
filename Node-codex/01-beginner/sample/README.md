# Beginner Sample API

## What This Covers
- Express routing and middleware
- Request validation and status codes
- File persistence with `fs/promises`

## Run
1. `npm install`
2. Copy `.env.example` to `.env` (optional)
3. `npm run dev`

Server: `http://localhost:3000`

## Endpoints
- `GET /health`
- `GET /tasks`
- `POST /tasks`
- `PATCH /tasks/:id`
- `DELETE /tasks/:id`
