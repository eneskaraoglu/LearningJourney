# Flutter Mini Projects

## Suggested Builds
- Task tracker
- Recipe browser
- Habit and streak app

## Expected Skills
- Multi-screen navigation
- Form validation
- API loading states
- Local persistence
- Shared theming

## Recommended Structure
- `lib/app/`
- `lib/features/`
- `lib/models/`
- `lib/services/`
- `lib/widgets/`

## Stretch Goals
- Add offline caching.
- Add a lightweight architecture layer.
- Add widget tests for the main flow.
