# Flutter Capstone Project

## Project Goal
Build a medium-sized mobile app that combines routing, forms, remote data, local persistence, theming, and testing.

## Suggested App Ideas
- Personal productivity manager
- Study planner
- Health habit tracker

## Required Capabilities
- Multi-screen flow
- Typed models and services
- Loading, empty, success, and error UI states
- Local persistence for preferences or drafts
- Reusable theme system
- Baseline tests for core flows

## Delivery Standard
- Clear feature-based folder structure
- Clean state ownership
- Predictable navigation
- Maintainable UI composition
- Readable test coverage for critical behavior

## Stretch Goals
- Offline-first behavior
- Push-ready notification architecture stub
- More advanced state pattern such as BLoC where justified
