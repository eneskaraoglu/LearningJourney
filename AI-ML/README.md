# AI-ML Learning Journey

This track is a practical roadmap to become an AI/ML professional with production-ready skills.

## Recommended Technology Stack

Core language and data stack:
- Python 3.11+
- NumPy
- pandas
- Matplotlib and Seaborn
- Jupyter Notebook and VS Code

Machine learning stack:
- scikit-learn
- XGBoost or LightGBM
- Optuna (hyperparameter tuning)

Deep learning stack:
- PyTorch
- torchvision and transformers

MLOps and deployment stack:
- FastAPI
- Docker
- MLflow
- DVC
- PostgreSQL
- GitHub Actions

LLM engineering stack:
- Prompt templates and output contracts
- Retrieval pipelines (TF-IDF and embedding-based)
- Evaluation harnesses (groundedness and relevance)
- Optional frameworks: LangChain or LlamaIndex

Cloud and serving options:
- AWS (S3, EC2, SageMaker) or GCP (GCS, Vertex AI)

## Outcome Goal

By finishing this track, you should be able to:
- clean and validate raw data,
- build baseline and advanced models,
- evaluate model risk and failure modes,
- deploy and monitor ML systems in production,
- build grounded LLM assistant workflows.
