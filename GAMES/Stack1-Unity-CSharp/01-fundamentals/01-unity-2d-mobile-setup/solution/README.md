# Solution - Unity 2D Mobile Setup

1. Create project with Unity LTS 2D template.
2. Build standardized folder structure in Assets.
3. Add bootstrap scene and scripts:
   - MobileBootstrap.cs
   - ProjectValidator.cs
4. Set Android package id and build development APK.
5. Run on device and capture checks:
   - Startup scene loads correctly.
   - Target FPS applied.
   - No startup errors in Console.
