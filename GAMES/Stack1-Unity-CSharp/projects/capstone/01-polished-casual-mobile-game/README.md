# Capstone - Polished 2D Casual Mobile Game (Play Store Ready)

## Goal
Ship one polished casual game with production-minded quality and Android release readiness.

## Scope
- Complete loop: onboarding -> gameplay -> rewards -> retry progression
- Robust state machine and decoupled UI
- Pooling and profiling-based optimization
- Audio mix and UX polish pass
- Save system for progress and settings

## Required Deliverables
- Gameplay design one-pager
- Architecture diagram (core systems and ownership)
- Android signed AAB for internal testing
- QA checklist with resolved critical bugs
- Store prep pack (icon, screenshots, short description, policy links)

## Done Criteria
- 10+ minutes stable play without blocking bugs
- No recurring GC spike causing visible stutter
- Crash-free test sessions on at least 3 Android devices
- Clear iOS port task list with risk notes
