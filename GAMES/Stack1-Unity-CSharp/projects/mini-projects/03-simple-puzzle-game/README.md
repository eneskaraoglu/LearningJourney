# Simple Puzzle Game (Mini Project 03)

## Concept
A grid-based tap puzzle where players clear tiles under move limits.

## Skills Practiced
- Grid data model and tile state transitions
- Input-to-grid coordinate mapping
- Scene management (menu, level, results)
- Save/load level progress with PlayerPrefs
- Lightweight animation and audio polish

## Completion Criteria
- At least 10 playable levels
- Move counter and win/lose states
- Retry and next-level flow
- Save unlocked level index
