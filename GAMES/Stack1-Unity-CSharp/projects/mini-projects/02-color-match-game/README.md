# Color Match Game (Mini Project 02)

## Concept
Player taps the lane that matches falling color targets before they cross a fail line.

## Skills Practiced
- Touch input hit zones
- UI score/combo feedback
- Spawn pacing and difficulty ramps
- Basic pooling for targets
- SFX feedback for correct/incorrect actions

## Completion Criteria
- 60-second gameplay loop
- Combo + multiplier system
- Miss penalty and game-over flow
- Best score persisted with PlayerPrefs
