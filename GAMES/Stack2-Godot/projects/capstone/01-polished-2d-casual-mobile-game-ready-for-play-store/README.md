# Capstone - Polished 2D Casual Mobile Game Ready for Play Store

## Scope
Deliver a production-structured casual game with stable Android release quality and clear iOS adaptation notes.

## Required Systems
- Clean architecture (game/application/infrastructure boundaries)
- Asset structure and naming strategy
- Game states: Boot, Menu, Playing, Paused, GameOver
- Settings panel with sound toggle
- High score save
- Performance pass (draw calls, allocations, startup time)

## Deliverables
- Architecture diagram + module map
- Core gameplay loop implementation
- Build checklist and release checklist
- Post-launch backlog for iOS + retention improvements
