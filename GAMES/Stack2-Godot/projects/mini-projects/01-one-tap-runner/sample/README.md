# Godot Starter (Runnable)

## Run
1. Open this folder as a Godot project.
2. Press F5 to run `scenes/main.tscn`.

## Android Build Steps
1. Install Android export templates from Godot editor.
2. Configure package name/version in Project Settings.
3. Set SDK/JDK paths in Editor Settings.
4. Export APK/AAB via Export Presets.

## iOS Notes
- iOS export requires macOS + Xcode.
- Validate safe area and performance on older devices.
