# Color Match Game

## Project Description
Build a simple all-ages 2D mobile game focused on short, replayable sessions.

## Core Systems
- Input command handling
- Player state and game state flow
- Collision/rule evaluation
- Score + high score persistence
- Audio and feedback signals

## Folder Structure Inside Engine
- Unity: `Assets/Scripts`, `Assets/Scenes`, `Assets/Prefabs`, `Assets/Art`, `Assets/Audio`, `Assets/UI`
- Godot: `scenes/`, `scripts/`, `art/`, `audio/`, `ui/`
- Flutter+Flame: `lib/game/`, `lib/ui/`, `lib/services/`, `lib/models/`
